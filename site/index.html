About Me

Infrastructure engineer with roots in the network layer

I'm a Cloud Infrastructure Engineer with hands-on experience designing, automating, and supporting AWS environments using Terraform and Linux-based systems. My background spans enterprise networking and systems administration across healthcare and banking — environments where reliability isn't optional.

I specialize in infrastructure as code, repeatable deployments, and operational efficiency. If something is done more than twice, it should be automated. I'm actively advancing toward professional-level AWS architecture and DevOps certifications, with CKA and AWS DevOps Professional on the roadmap for 2026.

Currently working as a Network Technician at Salina Regional Health Center while contracting as a cloud infrastructure engineer for IronClad Verify — and actively seeking full-time cloud and DevOps engineering roles.
